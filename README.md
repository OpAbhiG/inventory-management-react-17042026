# 🚀 Inventory & Assets Management System

A modern, scalable, and fully responsive **Inventory & Assets Management Web Application** built using **React 19 + Vite**.

This system helps organizations efficiently **track, manage, and analyze assets** with a clean and user-friendly interface.

---

## 📌 Overview

Managing assets manually becomes inefficient over time.  
This application provides a **centralized solution** to:

- Track all assets in one place  
- Monitor availability and usage  
- Visualize data with analytics  
- Improve operational efficiency  

---

## ✨ Key Features

- Modern UI with React 19  
- Fully responsive (Mobile, Tablet, Desktop)  
- Fast routing with React Router DOM  
- Interactive charts using Chart.js  
- Clean icons (FontAwesome + Bootstrap Icons)  
- API-ready architecture with Axios  
- High performance with Vite  
- Code quality with ESLint

## ⚙️ Getting Started

### 🔑 Prerequisites

- Node.js (v20 or above recommended)  
- npm  

---

### 📥 Installation

```bash
# Clone the repository
git clone <your-repo-url>

# Navigate into the project
cd inventory-management-react

# Install dependencies
npm install

# Start development server
npm run dev
```

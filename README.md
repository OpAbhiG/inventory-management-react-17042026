# Assets Management System

A modern and responsive **Inventory & Assets Management** web application built with React 19 and Vite.

This application helps businesses efficiently track, manage, and monitor their assets with a clean and user-friendly interface.

---

## ✨ Features

- Modern React 19 with latest features
- Fully Responsive Design using Bootstrap 5
- Smooth navigation with React Router DOM
- Interactive charts and analytics using Chart.js
- Beautiful icons with FontAwesome and Bootstrap Icons
- Ready for API integration using Axios
- Fast build and development with Vite
- Code linting with ESLint

---

## 🛠 Tech Stack

- **React** – 19.2.4

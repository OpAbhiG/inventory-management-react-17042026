🚀 Inventory & Assets Management System

A modern, scalable, and fully responsive Inventory & Assets Management Web Application built using React 19 + Vite.

This system is designed to help organizations efficiently track, manage, and analyze assets with a smooth user experience and clean UI.

📌 Overview

Managing assets manually becomes messy and inefficient over time.
This application provides a centralized digital solution to:

Track all assets in one place
Monitor usage and availability
Visualize data with analytics
Improve operational efficiency
✨ Key Features

✔️ Modern UI built with latest React
✔️ Fully responsive design (mobile + tablet + desktop)
✔️ Fast navigation using React Router
✔️ Interactive charts & analytics (Chart.js)
✔️ Clean icons with FontAwesome & Bootstrap Icons
✔️ API-ready architecture using Axios
✔️ Lightning-fast performance with Vite
✔️ Code quality enforced with ESLint

⚙️ Getting Started
🔑 Prerequisites

Make sure you have installed:

Node.js (v20 or above recommended)
npm or yarn
📥 Installation
# Clone the repository
git clone <your-repo-url>

# Navigate into the project
cd inventory-management-react

# Install dependencies
npm install

# Start development server
npm run dev
